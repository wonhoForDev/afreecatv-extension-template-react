# afreecatv-extension-template-react
vite-react template for afreecaTv extension

단지 기본 vite-react에 afreecaTv extension sdk와  
테스트 세팅, 심사용 옵션만 넣은 것 임  
각자 원하는게 다를 테니 기본적인 상태로만 구성 했음

## 개발 환경
- node 18.16.0
